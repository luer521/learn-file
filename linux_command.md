# Linux命令行

+ git init  初始化

+ vim+文件名 创建文件并打开编辑

+ git reflog  获取日志

+ git log 详细日志

+ git add  +文件名 将文件添加到缓存区

+ git commit -m “修改备注” 文件名 上传到本地库

+ git checkout 名字 切换分支

+ git branch 名字创建分支

+ git status 文件备选区

+ git merge 合并分支《如果冲突需要手动合并》

+ git remote 别名+网址 给网址创建别名

+ git remote -v查看当前别名

+ git pull/push/clone +网站/别名+当前用户分支名
